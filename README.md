# Personal Portfolio (vCard Style)

This project is a fully responsive personal portfolio website built using HTML, CSS, and JavaScript. It is designed to work seamlessly across desktop, tablet, and mobile devices.

## Installation

Download or copy the project files into your local environment.

Open the project in your browser:

```bash
open index.html
```

## Features

- Responsive design for all screen sizes  
- Clean and modern UI  
- Easy to customize  
- Lightweight and fast  

## Usage

You can customize the content, styles, and layout to fit your personal or professional needs.
